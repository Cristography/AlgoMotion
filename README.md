## نبذة

**AlgoMotion** منصة تعليمية مبنية بـ Vanilla HTML/CSS/JavaScript دون أي مكتبات خارجية، تُمكّن المطورين والطلاب من استيعاب الخوارزميات وهياكل البيانات من خلال تصورات مرئية تفاعلية خطوة بخطوة.

---

## الخوارزميات المدعومة

**خوارزميات الترتيب**

| الخوارزمية | أفضل حالة | متوسط | أسوأ حالة | الذاكرة |
|---|---|---|---|---|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |

**خوارزميات البحث**

| الخوارزمية | أفضل حالة | متوسط | أسوأ حالة | الذاكرة |
|---|---|---|---|---|
| Linear Search | O(1) | O(n) | O(n) | O(1) |
| Binary Search | O(1) | O(log n) | O(log n) | O(1) |

**شجرة البحث الثنائية (BST)**

| العملية | أفضل حالة | متوسط | أسوأ حالة |
|---|---|---|---|
| Insert | O(log n) | O(log n) | O(n) |
| Search | O(1) | O(log n) | O(n) |
| Delete | O(log n) | O(log n) | O(n) |

**القائمة المترابطة (Linked List)**

| العملية | أفضل حالة | متوسط | أسوأ حالة | الذاكرة |
|---|---|---|---|---|
| Insert | O(1) | O(n) | O(n) | O(1) |
| Search | O(1) | O(n) | O(n) | O(1) |
| Delete | O(1) | O(n) | O(n) | O(1) |

**المكدس والطابور (Stack & Queue)**

| العملية | أفضل حالة | متوسط | أسوأ حالة | الذاكرة |
|---|---|---|---|---|
| Push/Enqueue | O(1) | O(1) | O(1) | O(n) |
| Pop/Dequeue | O(1) | O(1) | O(1) | O(1) |
| Peek | O(1) | O(1) | O(1) | O(1) |

**الرسوم البيانية (Graphs)**

| الخوارزمية | أفضل حالة | متوسط | أسوأ حالة | الذاكرة |
|---|---|---|---|---|
| BFS | O(V+E) | O(V+E) | O(V+E) | O(V) |
| DFS | O(V+E) | O(V+E) | O(V+E) | O(V) |

## المميزات الرئيسية

- تصور مرئي تفاعلي مع إمكانية التنقل بين الخطوات يداً بيد
- تسليط الضوء على سطر الكود المُنفَّذ في الوقت الفعلي
- إحصائيات حية: المقارنات، التبديلات، وعدد الخطوات
- واجهة ثنائية اللغة (العربية / الإنجليزية) مع دعم RTL و LTR
- تصميم متجاوب يعمل على جميع أحجام الشاشات
- اختصارات لوحة المفاتيح: `Space` للتشغيل، `←` `→` للتنقل
- دعم كامل لـ: الترتيب، البحث، القوائم المترابطة، المكدس، الطابور، والرسوم البيانية
